Метод myWriter принимает в качетве аргумента строку сначала проверяет есть ли подходящий файл для записи данный методом isFileExists
если метод isFileExists вернет пустую строку будет создан файл с названием идентичный первому слову полученной строки.
После будет проверка есть в файле строка идентичный полученной, если есть будет выведена сообшение о том что такая запись есть
и метод завершит работу, иначе полученная строка будет записан в файл;

Метод isFileExists принимает в качестве аргумента массив файлов,
строку startsWith для поиска совпадений в начале названия и строку endsWith для поиска совпадений в конце строки
если аргумент находит совпадение то возврашает имя файла иначе возврашает пустую строку
